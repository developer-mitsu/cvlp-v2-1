Code Village

スパルタ教室Code Villageとは??

プログラミング教室Code Villageは、

「プログラミングを書けるようになりたい人を、書けるようにすること」

に価値をおき運営しております。

案件獲得保証もなければ、転職も確約しておりません。

稼ぐ為の特別なメソッドもございません。

しかし、どの選択肢も取れる様に最大限スキルを伸ばしたい、

そんな事ばかり考え熱中しております。

POINT1

1.圧倒的な学習量

プログラミングスキルを習得する最短の道は「圧倒的な学習を積む事」です。 私達が出来る事は生徒様に学習のマイルストーンを敷いて逸早く課題解決のサポートをする事です。 Code Villageでは1日3時間以上の学習を強要しており、学習の成果を毎日トレーナーに提出して頂きます。

POINT2

2.質問し放題

オフラインの講義外でも質問し放題です。優秀な講師陣がPCベタ張りで即レスを心がけております。テキストでの質問だけではなくSkypeを用いた課題解決が可能となっており、それは全てコンテンツプラットフォームにストックされ誰でもアクセス可能です。初学者の課題は重複するケースが多々あり、それを共有する事で空き時間に効率の良いインプットをする事が可能です。

POINT3

3.事前学習とレベル分け

受講開始二週間前に配布される全10回(約10時間)に及ぶ動画コンテンツを使用して初回講義までにHTML/CSSの基礎を習得して頂きます。そして7回目/全18回の講義ではクラス分けを実施し、それぞれのクラスに最適な講義を行ってまいります。努力した人にはそれだけの価値を届けたい思いでスクールにヒエラルキーを見出しました。

Code Village Online

オンライン学習プラットフォーム

オフラインの授業で学んだ事をCode Village Onlineを通じで復習しましょう。分かりやすい復習用ドキュメントはスマートフォンでも閲覧出来るので、隙間時間に効率よくインプットをする事が可能です。

また、毎週出される課題をダウンロードし取り組みましょう。 課題終了後はSNSや学習成果日報に知識をアウトプットしスキルを定着させます。

SNSや学習成果日報を用いた学習管理はコミットと継続を促し、更に早くスキル習得をする事が可能です。
