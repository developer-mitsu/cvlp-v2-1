スパルタ教室Code Villageとは??

プログラミング教室Code Villageは、

「プログラミングを書けるようになりたい人を、書けるようにすること」

に価値をおき運営しております。

案件獲得保証もなければ、転職も確約しておりません。

稼ぐ為の特別なメソッドもございません。

しかし、どの選択肢も取れる様に最大限スキルを伸ばしたい、

そんな事ばかり考え熱中しております。

POINT1

1.圧倒的な学習量

プログラミングスキルを習得する最短の道は「圧倒的な学習を積む事」です。 私達が出来る事は生徒様に学習のマイルストーンを敷いて逸早く課題解決のサポートをする事です。 Code Villageでは1日3時間以上の学習を強要しており、学習の成果を毎日トレーナーに提出して頂きます。

POINT2

2.質問し放題

オフラインの講義外でも質問し放題です。優秀な講師陣がPCベタ張りで即レスを心がけております。テキストでの質問だけではなくSkypeを用いた課題解決が可能となっており、それは全てコンテンツプラットフォームにストックされ誰でもアクセス可能です。初学者の課題は重複するケースが多々あり、それを共有する事で空き時間に効率の良いインプットをする事が可能です。

POINT3

3.事前学習とレベル分け

受講開始二週間前に配布される全10回(約10時間)に及ぶ動画コンテンツを使用して初回講義までにHTML/CSSの基礎を習得して頂きます。そして7回目/全18回の講義ではクラス分けを実施し、それぞれのクラスに最適な講義を行ってまいります。努力した人にはそれだけの価値を届けたい思いでスクールにヒエラルキーを見出しました。

02

Code Village

受講の流れ

FLOW1

事前課題に取り組む

入校前に事前課題としてhtml/cssの基礎理解を目指し、 簡易的な1枚のHPのコーディングが出来るまで何度も繰り返し実践して頂きます。 そして入校直前に渡されるテストモックを実装して頂き、知識の定着を確認します。

FLOW2

オフラインの講義を受講する

全18回の対面による集団講義を受講して頂きます。事前課題により基礎知識を習得した上での講義実施になりますので、より一層レベルの高いコンテンツ提供が可能になります。 出席出来なかった際は動画コンテンツ補講を受けて頂きます。

FLOW3

とにかく学習する

コンテンツプラットフォームを利用して課題に取り組み、隙間時間は質問ストックの視聴、 そしてインプットとアウトプットを行き来する学習法を用いてスキル習得を目指します。安心してください。質問はし放題です。

＋α

トレーナー制度

入プログラミングスキルを習得する目的は個人によって異なり、また受講中に変わる方も多く見受けられます。

それによって必要な学習量や、プログラミングスキル以外の学習要件も異なります。 そこでCode Villageではトレーナー制度を設け、都度最適なトレーナーが逐一アドバイスをさせて頂きます。

また、個々の学習状況、課題の進捗確認などを詳細に把握する事によって、 最適なアドバイス、マイルストーンの設計が可能になります。

＋α

教室開放

Code Villageでは講義以外の時間に自学習教室を解放しております。 残念ながら12名程しか入らない敷地面積になっておりますので完全予約制で利用して頂きます。

講師が常駐しておりますので勿論質問が可能です。 教室が空いていない場合はオンラインで質問が可能です。

＋α

オンライン質問

教室解放時だけではなく分からないことがあればオンラインでの質問も可能です。 全ての質問を映像で録画しており、他の生徒様も後にそれを閲覧する事が可能です。 生徒様の実際の課題ベースの動画コンテンツは空き時間に良質なインプットへと繋がります。

※質問対応時間:平日10:00~18:00

Code Village Online

オンライン学習プラットフォーム

オフラインの授業で学んだ事をCode Village Onlineを通じで復習しましょう。分かりやすい復習用ドキュメントはスマートフォンでも閲覧出来るので、隙間時間に効率よくインプットをする事が可能です。

また、毎週出される課題をダウンロードし取り組みましょう。 課題終了後はSNSや学習成果日報に知識をアウトプットしスキルを定着させます。

SNSや学習成果日報を用いた学習管理はコミットと継続を促し、更に早くスキル習得をする事が可能です。

事前課題サンプルダウンロード

初回講義二週間前に配布される全10回(約10時間)に及ぶ動画コンテンツです。 シンプルなWebサイトの制作を通して、 HTML/CSSを用いたマークアップの基礎を習得できます。 動画を見るだけではなく、実際に動画を適宜停止し手を動かしながら取り組んでいただきます。

そして入校直前に渡されるテストモックを実装して頂き、知識の定着を確認します。 基本的な技術を受講前に習得する事でより効率よくレベルの高い集団講義を実現します。
